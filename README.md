# craft-panel

A web panel for managing Minecraft servers on remote Linux hosts via SSH.

Built with Next.js, xterm.js, and SSH2. Supports Forge and Vanilla servers with full lifecycle management — file explorer, in-browser terminal, config editor with syntax highlighting, and server start/stop/install controls.

## Quick start

```bash
npm install
npm run dev
```
